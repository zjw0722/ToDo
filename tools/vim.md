第二讲


  1. 欲漠前光h除至巫/卧~末尾，入Udw

  2. 欲漠前光h除至前行末尾，入Ud$

  3. 欲h除整行，入Udd

  4. 在正常模式下一命令的格式是U

       [number]   command   object     或者     command [number]   object
     其意是U
       number - 代表的是命令绦械拇
       command - 代表要做的事情，比如 d 代表h除
       object - 代表要操作的ο螅比如 w 代表巫/卧~，$ 代表到行末等等。
                $ (to the end of line), etc.

  5. 欲撤消以前的操作，入Uu (小的u)
     欲撤消在一行中所做的改樱入UU (大的U)
     欲撤消以前的撤消命令，恢鸵郧暗牟僮鹘Y果，入UCTRL-R

  4. 更改命令的格式是U

         [number]   c   object        或者      c   [number]   object

第五讲
1. :!command 用于绦幸外部命令 command。

     看一些H例子U
         :!dir          -  用于@示前目的热荨
         :!rm FILENAME  -  用于h除名 FILENAME 的文件。

  2. :w FILENAME  可前 VIM 中正在的文件保存到名 FILENAME
     的文件中。

  3. :#,#w FILENAME 可前文件第 # 行至第 # 行的热荼４娴轿募
     FILENAME 中。

  4. :r FILENAME 可提取磁P文件 FILENAME K⑵洳迦氲疆前文件的光宋恢
     後面。
